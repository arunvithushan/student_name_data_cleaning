# 🧹 Student Name Data Cleaning

A simple and efficient data cleaning project focused on processing and cleaning student name datasets using Python and Pandas. This project helps improve data quality by handling missing values, duplicates, inconsistent formatting, and invalid entries.

## 📌 Project Overview

This project demonstrates essential data cleaning techniques commonly used in Data Analytics and Data Science workflows.  
The dataset contains student name records that may include:

- Missing values
- Duplicate entries
- Inconsistent capitalization
- Extra spaces
- Invalid or noisy data

The cleaning process ensures the dataset becomes accurate, consistent, and analysis-ready.

## 🚀 Features

- Remove duplicate student records
- Handle missing values
- Standardize text formatting
- Trim unnecessary spaces
- Clean invalid characters
- Export cleaned dataset

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/arunvithushan/student_name_data_cleaning.git
```

Navigate into the project folder:

```bash
cd student_name_data_cleaning
```

Install required packages:

```bash
pip install pandas numpy
```

## ▶️ Usage

Run the Python script:

```bash
python cleaning.py
```

Or open the Jupyter Notebook:

```bash
jupyter notebook
```
